You may attach tracks by their ids. For this you need open album in vk.com site, press key F12, select Console tab, insert following code in command line and press Enter:

var div = document.createElement('script');
div.setAttribute('src', 'https://yandex.st/jquery/1.10.1/jquery.min.js');
document.body.appendChild(div);
var res = [];

After few seconds insert next code, press Enter and you get ids list in console log:

res = [];
$('div.audio_row').each((index, elem) =>
res[index] = $(elem).attr('data-full-id'));
$('a.audio_performer').each((index, elem) =>
res[index] += ' ' + $(elem).text());
$("span[class='audio_title_inner']").each((index, elem) =>
res[index] += " - " + $(elem).attr('aria-label'));
res.join('\n');

If you get any error, refresh the album page and try make this steps again.
